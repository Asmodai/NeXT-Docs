NXImageView Styles

DECLARED IN eointerface/NXImageView.h

SYNOPSIS    typedef enum {
NX_ImageNoFrame = 0,
NX_ImagePhoto,
NX_ImageGrayBezel,
NX_ImageGroove
} NXImageStyle;

DESCRIPTION NXImageStyle defines the possible borders an NXImageView draws around itself. The table below describes the border styles; you can also examine the four styles in Interface Builder.

Value Description
NX_ImageNoFrame No border
NX_ImagePhoto Light gray line surrounding image, with a drop shadow
NX_ImageGrayBezel Bezeled well
NX_ImageGroove Grooved line




Symbolic Constants




NXTableView Constants

DECLARED IN eointerface/NXTableView.h

SYNOPSIS    NX_NOSELECT
NX_NoIndex

DESCRIPTION These two constants are used by the NXTableView class. NX_NOSELECT is a selection mode along the lines of those defined by the Matrix class. If an NXTableView's selection mode is NX_NOSELECT then no selection at all is permitted.

NX_NoIndex is used to indicate that nothing in the NXTableView is selected.
