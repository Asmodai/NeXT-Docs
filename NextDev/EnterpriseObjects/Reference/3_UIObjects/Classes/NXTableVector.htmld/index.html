NXTableVector



Inherits From: Object
Conforms To: NXTableVectors
Declared In: eointerface/NXTableVector.h



Class Description

An NXTableVector maintains the formatting information for an individual row or column of an NXTableView. See the NXTableVectors protocol specification for more information on formatting information.



Instance Variables

id identifier;

id formatter;

NXCoord minSize;

NXCoord maxSize;

NXCoord currentSize;

char *title;

id titleFont;


identifier The object identifying the attribute displayed in the vector.
formatter The vector's NXFormatter.
minSize The vector's minimum height (for a row vector) or width (for a column vector).
maxSize The vector's maximum height or width.
currentSize The vectors current height or width.
title The vector's title.
titleFont The Font for the vector's title.
