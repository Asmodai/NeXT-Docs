EOExpressionContext



Adopted By: EOSQLExpression
Incorporates: NSObject
Declared In: eoaccess/EOExpressionArray.h



Protocol Description

The EOExpressionContext protocol declares methods for proper aliasing of references to attributes within SQL statements. An object that conforms to this protocol is called an expression context object. The "context" of an expression context object consists of its current data path, relative to which all attributes are aliased, and a set of aliases used for the entities that the attributes belong to.



Instance Methods

expressionValueForAttribute:
(NSString *)expressionValueForAttribute:(EOAttribute *)anAttribute

Returns a string representation of anAttribute suitable for use in a SQL statement. Any data paths associated with anAttribute are aliased, in a form such as "t0.l_name".

See also:  columnName (EOAttribute)



expressionValueForAttributePath:
(NSString *)expressionValueForAttributePath:(EOExpressionArray *)path

Returns a string representation of path, aliased for use in a SQL statement. path is a data path ending in an attribute. Records the relationships up to the attribute and registers an alias used by the context for all future references to that particular relationship path (doesn't register an alias if one already exists). This alias is prepended to the attribute's column name, in a form such as "t0.l_name".
