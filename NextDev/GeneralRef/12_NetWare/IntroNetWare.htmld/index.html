Introduction

NEXTSTEP Release 3 provides programming support for Novell NetWare networking, allowing your application to communicate with other applications on networked PCs, and to access printers on those networks.  The NetWare reference documentation is provided by Novell, and may be found online in /NextLibrary/Documentation/Novell.
