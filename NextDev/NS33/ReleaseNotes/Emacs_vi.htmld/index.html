3.3 Release Notes:
Emacs and vi




This file contains information about the Emacs and vi text editors.  There are no specific notes for Release 3.3.


Bugs Fixed in Release 3.2

Reference: 33176
Problem: Emacs and vi didn't work in console mode on Intel platforms.
Description: When you used the console on an Intel-architecture machine (by typing "console" at the login window), you couldn't use emacs or vi to edit files. This has been corrected.
