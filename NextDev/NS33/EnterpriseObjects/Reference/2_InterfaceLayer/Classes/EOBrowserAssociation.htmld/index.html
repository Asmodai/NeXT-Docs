EOBrowserAssociation



Inherits From: EOAssociation : NSObject
Conforms To: EOAssociationNotification (EOAssociation)
Declared In: eointerface/EOBrowserAssociation.h



Class Description

An EOBrowserAssociation manages and monitors a single-column NXBrowser (more accurately, the left-most column of any NXBrowser). It uses the browser to display the values of enterprise objects for its key, and alerts its controller of changes in the browser's selection. You can thus use a browser association to implement a selection list, so that when the user selects an enterprise object's entry in the browser, other user interface objects display values for other properties of the selected enterprise object.

Although the interface of any association contains only a few methods, various implementations are often highly dependent on one another and on the way the destination objects act. Because of this, if you need to create an association class for a custom subclass of NXBrowser, you should create it as an immediate subclass of EOAssociation.
