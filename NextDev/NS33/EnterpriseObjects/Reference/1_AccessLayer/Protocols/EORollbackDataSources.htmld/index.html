EORollbackDataSources



Adopted By: EODatabaseDataSource
EODetailDatabaseDataSource
Incorporates: EODataSources
Declared In: eoaccess/EODataSources.h



Protocol Description

The EORollbackDataSources protocol declares the rollback method for reverting changes. A data source object (as described in the EODataSources protocol specification) indicates its ability to reverse changes by adopting this protocol.



Instance Methods

rollback
(void)rollback

Reverses any changes made to the data source with insertObject:, deleteObject:, or updateObject: since the receiver was last sent a saveObjects message (these methods are all declared by the EODataSources protocol). This method affects only the data source's storage--not the enterprise objects in memory.
