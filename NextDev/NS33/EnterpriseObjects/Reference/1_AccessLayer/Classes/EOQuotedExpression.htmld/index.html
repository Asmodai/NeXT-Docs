EOQuotedExpression



Inherits From: NSObject
Conforms To: NSCopying
Declared In: eoaccess/EOQuotedExpression.h



Class Description

An EOQuotedExpression is used to quote another object's expression value (as defined in the EOExpression informal protocol specification). It's used internally by the Framework in the generation of SQL; unless you're implementing an adaptor, you should never need to create or use one yourself.

When asked for its expression value, an EOQuotedExpression returns an NSString containing the expression value of the object it was created with, enclosed on each end by its quotation string. Also, any occurrence of the quotation string within the expression value is preceded with an escape string. For example, if an EOQuotedExpression is created with the following items:

expression value Tom's Diner
quote string '
escape character '

its expression value is:

'Tom''s Diner'
