EONull



Inherits From: NSObject
Conforms To: NSCopying
Declared In: eoaccess/EONull.h



Class Description

An EONull object represents a NULL value on the database server. When you fetch rows with an EOAdaptorChannel or enterprise objects with an EODatabaseChannel, any attribute of that row or object with a NULL value on the server will have an EONull object as its value.

EONull generates a single unique instance for any running application. You should check whether a value is NULL by using pointer equality:

if (myValue == [EONull null]) {
/* ... */
}
