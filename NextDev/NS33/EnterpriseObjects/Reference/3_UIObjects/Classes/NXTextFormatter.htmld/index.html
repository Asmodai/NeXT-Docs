NXTextFormatter



Inherits From: NXFormatter : Object
Declared In: eointerface/NXTextFormatter.h



Class Description

An NXTextFormatter displays read-only text values in an NXTableView. This class displays text more quickly than the companion NXEditableTextFormatter class. See the NXFormatter class specification for more information.
