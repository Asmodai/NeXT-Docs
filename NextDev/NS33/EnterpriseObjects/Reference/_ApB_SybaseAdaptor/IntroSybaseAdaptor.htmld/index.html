Introduction

The adaptor-level classes define a server-independent interface for working with relational database systems. The Sybase adaptor provides subclasses for exploiting the unique features of Sybase database servers, while supporting the generic interface common to all adaptors.




Classes

The SybaseAdaptor class provides methods for sharing DBProcesses and configuring the maximum textsize. It also provides Sybase-specific message and error handling, as does the SybaseChannel class. The SybaseChannel class includes delegate methods for error and message handling, as well as delegate methods that support the processing of compute rows and stored procedures. The SybaseContext class provides a method for initializing a context from a SybaseAdaptor object.
