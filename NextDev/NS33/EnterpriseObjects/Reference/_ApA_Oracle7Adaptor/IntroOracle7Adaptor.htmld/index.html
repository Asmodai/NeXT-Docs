Introduction

The adaptor-level classes define a server-independent interface for working with relational database systems. The Oracle7 adaptor provides subclasses for exploiting the unique features of Oracle7 database servers, while supporting the generic interface common to all adaptors.




Classes

The OracleAdaptor class provides a method for returning the connection dictionary components as an NSString. The OracleChannel class provides methods for setting the fetch buffer length and handling errors. Both the OracleChannel and OracleContext classes provide methods for returning information about Oracle-specific data structures that can be used by the channel delegate to determine the cause of an error.
