This directory has an example server, together with a client that uses it.  Chapter 2, "Using Mach Messages," of NeXT Operating System Software uses this example to explain how to write MiG servers.

To build the add server and a sample client, follow these steps:

1)  Make a copy of this directory; then change to the new directory at a shell prompt.  For example:
localhost> cp -r /NextLibrary/Documentation/NextDev/Examples/MiG ~
localhost> cd ~/MiG

2)  Run make in the Server directory.  (It's important that you build the server before the client.)
localhost> cd Server
localhost> make

3)  Run make in the Client directory.
localhost> cd ../Client
localhost> make



To use the add server:

1)  From the MiG directory, run the server in the background.
localhost> cd ..
localhost> addserver&

2)  Run the client once for each call you want to make to the server.
localhost> addclient 81 1
According to the server, 81 + 1 = 82.

3)  Kill the server when you're done using the running version of it.
localhost> jobs
[1]  + Running              addserver
localhost> kill %1
