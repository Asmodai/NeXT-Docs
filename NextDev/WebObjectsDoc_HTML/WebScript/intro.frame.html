Using WebScript

This chapter provides an overview of WebScript, the WebObjects scripting language.This chapter is divided into two major sections:

  • The WebScript Language

This section decribes basic WebScript language syntax.

  • Using WebScript in a WebObjects Application

This section describes using WebScript within the context of a WebObjects application. It uses a simple example application to explain the issues that arise in creating a WebObjects application, as well as special WebScript features.

For a detailed discussion of the structure of a WebObjects application, see Getting Started.
