Neat-o new stuff

Look what's here just in time for the holiday season--NEXTSTEP Release 3.2 and the latest issue of NEXTSTEP In Focus!

There's lots to be excited about in Release 3.2: new drivers and Driver KitTM, a bundled demo of Insignia Solutions' SoftPC®, improvements to some developer tools, and bug fixes galore, for a start. "What's New in Release 3.2?" outlines these and other new features in both the user and developer systems. "The New SoftPC" highlights what you can do with the new Intel version of SoftPC.

Once you've upgraded you might want to make sure your network works as well as it should. "NFS Performance Tuning" explains how to tune NFS for your network; in particular, it focuses on getting optimal performance over PC Ethernet adapters. Then "Advanced Networking Tools" covers tools for monitoring and improving performance of large networks.

But maybe you just want to set up your PC with some SCSI devices. No sweat--"SCSI in a PC World" shows you the basics and highlights differences between using SCSI devices on workstations and PCs.

Have still more questions? Our new regular department, "Frequently Asked Questions," gives answers to the questions our Support team answers most. "Tips & Techniques" and "FYI" present other interesting tidbits, like the new syntax for boot commands and a list of excellent administration reference books.

--Cynthia Closkey, Editor
