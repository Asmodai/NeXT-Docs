3.3 Release Notes:
The DOS Filesystem




This file contains release notes for the 3.3 release of the DOS filesystem.



Notes Specific to Release 3.3


New Features

The following new feature has been added to the DOS filesystem since Release 3.2.

Support for reading and writing removable DOS hard drives.


Bugs Fixed in Release 3.3

These bugs have been fixed in Release 3.3

Using large DOS hard drives often led to erroneous data when reading and reporting disk information, and led to corruption when writing.  These corruption issues have been fixed.
DOS filesystem performance has been improved significantly under many circumstances.
  Under a number of circumstances in 3.1 and 3.2, certain DOS partitions wouldn't mount.  All known instances of this problem have been addressed.
If you tried to use a floppy that had been formatted with Norton Safe Format, the disk label used to be displayed incorrectly.
Sometimes, DOS wasn't listed as a possible option in the workspace initialization panel.
