3.3 Release Notes:
The SimpleNetworkStarter Application




This file contains release notes for the SimpleNetworkStarter application.




Notes Specific to Release 3.3


New features

When it starts up, SimpleNetworkStarter creates backup copies of /etc/netinfo and /etc/hostconfig in /etc/netinfo.backup and /etc/hostconfig.backup.
During startup, SimpleNetworkStarter checks the system and network configuration. This may take a few moments.
SimpleNetworkStarter keeps a log of its activities. You can examine log messages in its Log View. A duplicate copy of the log is written to the file /usr/adm/SNS.log. This log file is cleared each time you start up SimpleNetworkStarter. The log includes a report of the network and system configuration that SimpleNetworkStarter determines at startup as well as a report of all configuration changes that it makes when you configure a computer.
You can use SimpleNetworkStarter to undo or change an existing configuration.
Support for simple SNMP configuration has been added.
You can configure new NetInfo clone servers on new subnets using the NetInfo Options panel. This panel is enabled only if the computer is on a subnet that has no servers for a parent NetInfo domain. In such a case, you can configure the computer to run a clone NetInfo server of a master on another subnet.


Changes

SimpleNetworkStarter no longer checks memory size and disk space availability for various server configurations.
